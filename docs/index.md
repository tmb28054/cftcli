## Overview

CFT CLI provides command-line tools to ease interaction with AWS CloudFormation while developing and testing CloudFormation stacks.

## Features

- Create, update, and delete CloudFormation stacks
- Monitor stack operations with real-time status updates
- Manage stack policies and termination protection
- List and describe stacks and pipelines
- Execute CodeBuild projects
- Parameter file support (JSON/YAML)
